相对定位relative

relative

position: relative; left: 30px; top: 20px;

relative

从效果看，相对定位是相对前一个元素没有偏移前的位置的定位。 或者换个说法，相对定位的元素在dom中占用的位置是其原本应该在的位置，而不受偏移值的影响

fixed

fixed

从效果看，不会受页面滚动，留在视察的固定位置

绝对定位absolute

从使用效果看，使用绝对定位时，元素实际上不占用空间，但是显示的时候根据margin, left, right等值调整位置。
下面的这条粗线实际上是因为div元素内没有实际上的元素占位，所以border-top跟border-bottom合在一起
